This entire application is running without Zone.js. All UI updates are driven by signals.

Why it matters

  • Smaller Bundle: No need to ship the Zone.js library, reducing the application's footprint.
  • Performance: Eliminates the overhead of Zone.js's patching of browser APIs and the associated microtask churn.
  • Predictable Reactivity: Change detection only runs when a signal is updated, giving developers explicit control over when the UI re-renders.
  • Future-Proof: Aligns Angular with modern JavaScript frameworks and browser standards.

How to verify

  1. Confirm that the "Zoneless: ON" badge is visible in the header HUD.
  2. Observe that the timer updates every second. This is driven by a signal updated from a native setInterval, which Zone.js would normally patch.
  3. Click the "Native Listener" button. The counter updates, proving that native DOM events also trigger re-renders through signals.
